# TK1 B3 Submission Package

Paket ini disusun untuk pengumpulan `Tugas Kelompok 1` dan dibagi menjadi dua folder utama:

- `Soal 1`: implementasi Python dan hasil eksperimen untuk `Block LU` dan `Thomas 4-diagonal` pada data asli ukuran `8 x 8` sampai `512 x 512`.
- `Soal 2`: implementasi MATLAB untuk pemodelan lintasan bintang dengan `Normal Equation` dan `QR Householder`.

Laporan gabungan final disertakan di root paket dengan nama:

- `TK1_B3_2106652682_2406352424_2406355893_2406434140.pdf`

Untuk `Soal 1`, file yang paling cepat dibuka bila ingin melihat hasil akhir adalah:

- `Soal 1/Hasil/ringkasan.csv`

File tersebut memuat ringkasan runtime, memory, CPU/GPU usage, condition number, dan error untuk kedua metode. Solusi `x_b` dan `x_c` per ukuran juga disediakan di folder `Soal 1/Hasil/` dalam bentuk `.txt`.
